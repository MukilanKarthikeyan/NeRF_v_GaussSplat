# Project 6: NERF

## Getting Started

Pre-requirements:

jupyter notebook
relevant nvidia cuda drivers
conda installation
  
  - Ensure that you are using the environment `nerf_gauss`
  1. make sure you are in your main directory
  2. ```conda env create -f environment.yml```
  3. ```conda activate nerf_gauss```



Dataset: Please refere to the [NeRF dataset](https://www.matthewtancik.com/nerf)
